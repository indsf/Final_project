# 4242 도움받는 사이 도움주는 사이
## 장애인 일자리 추천 및 도우미 매칭 시스템
### 개요
- 프로젝트 기간 :2025년
- 프로젝트 참여자 : 이인호,김두한
- 사용 기술 : **spring,MySQL**

---

### 프로젝트 개요
**장애인의 이동 불편과 취업 장벽을 낮추는 AI 기반 맞춤 지원 플랫폼**♿

- 대구대 캠퍼스·식당·공공기관 등 협소한 실내 공간과 복잡한 동선으로 인해 휠체어·보행 보조기 이용자는 일상적인 이동에 불편을 겪음

- 도우미 지원 제도가 존재함에도 인력 부족과 실시간 매칭 비효율로 실제 체감 지원은 낮은 상황

- 취업 단계에서는 장애인 맞춤 채용공고 탐색이 어렵고 근무환경·접근성·유연근무 등 실질적 조건 정보가 부족해 구직 장벽이 지속됨

- 생활 지원의 즉시성 부족, 접근성 정보의 비가시성, 개인 맞춤 일자리 추천 부재 문제

- 이를 해결하기 위해 AI를 활용한 실시간 도움 매칭과 맞춤형 일자리 추천 서비스를 통해 장애인의 일상과 취업의 불편을 동시에 줄이고자 함
---
### 주요 기능

#### 1. 회원가입 및 로그인
- 사용자 회원가입 및 로그인 기능 구현
- 사용자 정보를 기반으로 개인 맞춤형 서비스 이용 가능

#### 2. 도움 요청 / 도움 제공 게시글 작성
- 도움이 필요한 사용자와 도움을 주고자 하는 사용자가 각각 게시글 작성 가능
- 도움 유형, 일정, 장소, 상세 내용 등을 입력하여 게시글 등록

#### 3. 댓글 기반 매칭 시스템
- 게시글의 댓글 기능을 통해 요청자와 지원자가 연결됨
- 매칭 수락 시 도움 요청자와 지원자 간 매칭 성립

#### 4. 실시간 채팅 기능
- 매칭된 사용자 간 실시간 채팅 가능
- 채팅을 통해 세부 일정 및 지원 내용을 조율할 수 있도록 구현

#### 5. 마이페이지
- 내 정보 조회
- 내가 작성한 게시글 확인
- 매칭 진행 현황 및 완료 내역 관리

#### 6. AI 기반 장애인 맞춤 일자리 추천
- 사용자의 지역, 직무 등의 조건을 반영한 맞춤형 일자리 추천
- 학습된 추천 모델을 활용하여 적합한 채용공고 제공

#### 7. 데이터베이스 기반 통합 관리
- 회원, 게시글, 매칭, 채팅, 추천 관련 데이터를 데이터베이스에 저장 및 관리
- 서비스 전반의 사용자 활동 내역을 체계적으로 관리 가능

---

### 기대 효과
- 장애인과 지원자를 빠르게 연결하여 실질적인 생활 지원 제공
- 장애인 맞춤형 일자리 정보를 제공하여 취업 접근성 향상
- 일상 지원과 취업 지원을 통합한 서비스로 사용자 편의 증대
